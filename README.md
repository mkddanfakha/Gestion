# Gestion

Application de gestion développée avec Laravel et Vue.js.

## Installation

1. Cloner le dépôt
2. Installer les dépendances : `composer install` et `npm install`
3. Copier `.env.example` vers `.env` et configurer
4. Générer la clé d'application : `php artisan key:generate`
5. Exécuter les migrations : `php artisan migrate`
6. Construire les assets : `npm run build`

## Technologies

- Laravel 12
- Vue.js 3
- Inertia.js
- Bootstrap 5
- TypeScript
